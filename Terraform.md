# Terraform

Running old terraform website locally with Docker

## Terraform Website

### Version 0.11.4

```bash
docker run --rm -p 3000:3000 ghcr.io/praveenprem/terraform-website:0.11.4
```

## AWS Provider

### Version 0.11

```bash
docker run --rm -p 4567:4567 ghcr.io/praveenprem/docs/terraform-aws-docs:0.11
```
